# Binary Search Tree (BST) Implementation

## Description

A Java-based Binary Search Tree (BST) implementation developed as part of the Java Development Internship at Codveda Technologies.

This project demonstrates core Data Structures and Algorithms concepts including insertion, deletion, searching, and tree traversals using Object-Oriented Programming principles.

## Features

- Insert Nodes
- Delete Nodes
- Search Nodes
- Inorder Traversal
- Preorder Traversal
- Postorder Traversal
- Object-Oriented Design

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Data Structures
- Algorithms
- Recursion

## How to Run

Compile:

```bash
javac TreeNode.java BinarySearchTree.java BSTDemo.java
```

Run:

```bash
java BSTDemo
```
